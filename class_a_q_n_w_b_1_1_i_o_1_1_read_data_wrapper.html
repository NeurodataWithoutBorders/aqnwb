Detailed Description

template<StorageObjectType OTYPE, typename VTYPE = std::any>
class AQNWB::IO::ReadDataWrapper< OTYPE, VTYPE >

Class for wrapping data objects (datasets or attributes) for reading data from a file.

Template Parameters
OTYPEThe type of object being wrapped defined via AQNWB::Types::StorageObjectType
VTYPEThe data type of the values stored in the data object

Reads an dataset with a specified data type.

This convenience function uses valuesGeneric to read the data and then convert the DataBlockGeneric to a specific DataBlock/

We do not support slicing for attributes, so this function is disabled for attributes. For attributes we should only use the valuesGeneric() method without arguments.

Template Parameters
Tthe value type to use. By default this is set to the VTYPE of the object but is added here to allow the user to request a different type if approbriate, e.g., if the object uses VTYPE=std::any and the user knows the type VTYPE=float
Parameters
startThe starting indices for the slice (optional).
countThe number of elements to read for each dimension (optional).
strideThe stride for each dimension (optional).
blockThe block size for each dimension (optional).
Returns
A DataBlock structure containing the data and shape.
