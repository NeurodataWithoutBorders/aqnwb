std::shared_ptr< ElectrodesTable > NWBFile::createElectrodesTable ( std::vector< Types::ChannelVector > recordingArrays,
bool finalizeTable = true )

Create ElectrodesTable. Note, this function will fail if the file is in a mode where new objects cannot be added, which can be checked via nwbfile.io->canModifyObjects().

Parameters
recordingArraysvector of ChannelVector indicating the electrodes to add to the table. This vector should contain all the electrodes that are detected by the acquisition system, not only those being actively recorded from.
finalizeTableIf true (default) then the table will be finalized after creation to write it to the file. If false, the caller must call finalize() on the returned table object to write it to the file.
