Detailed Description

template<typename DTYPE, std::size_t NDIMS>
class AQNWB::IO::ConstMultiArrayView< DTYPE, NDIMS >

Non-owning, multi-dimensional, read-only array view for contiguous data.

Provides multi-dimensional access to a flat data buffer using row-major order. This class is used by DataBlock::as_multi_array for C++17/20 compatibility.

Template Parameters
DTYPEThe data type of the array elements.
NDIMSThe number of dimensions.
Note
For C++23 and later, prefer using std::mdspan for multi-dimensional array access.
This class may be deprecated and removed in future releases in favor of std::mdspan.
