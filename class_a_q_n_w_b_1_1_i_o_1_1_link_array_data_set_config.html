Detailed Description

Configuration for creating a soft-link to an existing dataset.

This class allows configuration of a dataset as a soft-link (symbolic link) to another dataset in the file, avoiding data duplication. This is useful for scenarios like time-alignment where multiple TimeSeries share the same data but have different timestamps.
